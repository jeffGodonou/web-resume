# Jeff Godonou — Web Portfolio

A personal portfolio and web résumé built with Next.js and Tailwind CSS, designed to showcase projects, skills, and professional experience.

---

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** JavaScript
- **Styling:** Tailwind CSS
- **Linting:** ESLint
- **Font:** Geist (via `next/font`)

---

## Project Structure

```text
web-resume/
├── public/         # Static assets (images, icons)
└── src/app/        # Next.js App Router pages and components
```

---

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Run locally

```bash
# Clone the repo
git clone https://github.com/jeffGodonou/web-resume.git
cd web-resume

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be available at `http://localhost:3000`

### Build for production

```bash
npm run build
npm start
```

---

## Deployment

This project is optimized for deployment on [Vercel](https://vercel.com). Connect the GitHub repo to Vercel and it will auto-deploy on every push to `main`.
